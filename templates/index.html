Best Practices

Prompt Engineering Do's & Don'ts

✅ Do These
  • ✓Be specific about requirements and constraints
  • ✓Provide examples and context in your prompts
  • ✓Specify your tech stack and versions explicitly
  • ✓Mention edge cases and error scenarios
  • ✓Request error handling and loading states
  • ✓Ask for mobile-first responsive design
  • ✓Include accessibility requirements upfront
  • ✓Define data validation rules clearly
❌ Avoid These
  • ✗Vague requests like "make it better"
  • ✗Requesting too many features at once
  • ✗Ignoring the existing code structure
  • ✗Skipping testing requirements entirely
  • ✗Forgetting about mobile and tablet users
  • ✗Overlooking security considerations
  • ✗Neglecting error handling scenarios
  • ✗Not specifying performance expectations
⚡ Power Tips
  • →Iterate one feature at a time for best results
  • →Share error messages verbatim, not paraphrased
  • →Give AI the full file context, not just the problem area
  • →Reference existing components for consistency
  • →Ask for explanations alongside code changes
  • →Build the happy path first, then edge cases
  • →Use version control before major AI-driven changes
  • →Test each iteration before adding more complexity

How to Use This Sandbox

Complete guide to generating and using AI prompts effectively.

📖

Step 1 — Open Sandbox

Navigate to the Sandbox tab. You'll see 8 industry categories on the left. Each industry has 5 curated design themes and 20 pre-selected pages.

🏭

Step 2 — Pick Industry

Click any industry — Healthcare, E-Commerce, Fintech, EdTech, SaaS, Real Estate, Social, or Logistics. The UI updates immediately with relevant options.

🎨

Step 3 — Choose Theme

Each industry has 5 professional design themes. Click a theme pill to select it. This changes the visual personality of the entire app in the generated prompt.

📄

Step 4 — Select Pages

Choose which of the 30 pages to include. All are selected by default. Deselect pages you don't need or click "Select All" to include everything.

⚡

Step 5 — Generate Prompt

Click "Generate Prompt" to produce a complete, production-ready AI prompt covering your industry, theme, pages, tech stack, and all requirements.

📋

Step 6 — Copy & Use

Click "Copy Prompt" and paste it into any AI assistant (Claude, ChatGPT, Gemini, etc.). The AI will build your complete app from the detailed specification.

🔄

Iterate with Templates

After the initial build, use the Templates tab for iteration prompts — adding features, fixing bugs, improving UI, integrating APIs, and deploying.

🌗

Dark / Light Mode

Toggle between dark and light mode using the moon/sun button in the navbar. Your preference is saved and persists across visits.

💡

Pro Tip

Start with a focused subset of pages (5–10), get them working perfectly with the AI, then come back and add more pages with the template iteration prompts.
